## Project

Student Performance Analyzer

The program:
- Stores student marks in an array
- Calculates total marks
- Calculates average marks
- Finds the highest mark
- Determines the result using a ternary operator

## Key Learning

Arrays help store multiple values in a single variable and loops help process them efficiently.
